BRAIN-GAMES is a collection of simple math games. This is my first Python project.
It includes 5 games:
1. Brain-even
2. Brain-calc
3. Brain-gcd
4. Brain-progression
5. Brain-prime

PROJECT DEPENDENCIES:

python = >=3.8.1,<4.0
prompt = ^0.4.1
flake8 = ^7.0.0
